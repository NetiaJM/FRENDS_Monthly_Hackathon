### specification
* bookmarklet
	1. 북마클릿을 추가한다.
	2. 특정 사이트로 접속한다.
	3. 북마클릿을 실행한다.
	4. 좋다, 나쁘다를 선택한다.
	5. 로컬 스토리지에 값을 날짜와 시간을 포함해 저장한다.
	6. 3항에서 접속중인 사이트에 이미 좋다, 나쁘다에 대한 평가값이 있다면 표시한다.
